# Telegram Tobacco Shop Bot

Этот проект содержит простого Telegram бота с Mini App (Web App) для продажи табака.

## Структура
- `bot.py`: Основной файл бота на Python (aiogram).
- `webapp/`: Папка с веб-приложением.
  - `index.html`: Главная страница.
  - `styles.css`: Стили.
  - `script.js`: Логика приложения.

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Настройка Web App

Telegram Mini Apps требуют, чтобы веб-приложение было размещено на HTTPS сервере.
Вы **не можете** использовать локальный путь (например, `file://...`) напрямую.

Варианты размещения:
1. **GitHub Pages**: Загрузите папку `webapp` в репозиторий GitHub и включите Pages.
2. **Vercel / Netlify**: Просто перетащите папку `webapp` в панель управления.
3. **Ngrok** (для тестов): Запустите локальный сервер и создайте туннель.

### После размещения:
1. Откройте `bot.py`.
2. Найдите переменную `web_app_url`.
3. Замените её на вашу ссылку (например, `https://your-site.com/webapp/index.html`).

## Запуск бота

```bash
python bot.py
```
